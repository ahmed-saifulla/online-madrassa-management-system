# Online Madrassa Management (Starter)

Starter Next.js + Tailwind scaffold with a responsive Nav and a Teachers list page.

Quick start

```bash
npm install
npm run dev
```

Open http://localhost:3000 and go to the Teachers page.

Files of interest

- components/Nav.js
- pages/teachers/index.js
- data/teachers.js
